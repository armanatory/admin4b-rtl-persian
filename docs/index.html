پیشگفتار

Admin 4B یک قالب مدیریتی متن‌باز بر اساس بوتسترپ ۴ است. سورس‌کد نگارش پارسی و راست‌چین شده‌ی این قالب از مخزن گیت‌هاب قابل دسترسی می‌باشد.

آغاز سرعتی!

این قالب از بسیاری امکانات بوتسترپ۴ بهره جسته، پس چنانچه می‌خواهید از بیشتر گزینه‌های این قالب بهره بجویید، بایستی با امکانات و گزینه‌های بوتسرتپ۴ آشنا باشید. پیشنهاد می‌شود که پیش از آغاز کار، سری به وب‌سایت بوتسترپ و مستندات آن بزنید.

برای اینکه قالب به‌خوبی کار کند باید پیش‌نیازهای بوتسترپ۴ شامل جی‌کوئری و Popper و همچنین فایل css کتابخانه‌ی فونت‌آسام و آیکن‌های ساده‌ی خطی در پروژه‌ی پیوست شوند. سپس آخرین نگارش قالب مدیریتی/پیشخوان Admin 4B را دریافت نمایید.

فایل‌های بوتسترپ ۴

لازم نیست که فایل‌های بوتسترپ‌ را در پروژه فراخوانی کنید چرا که آن‌ها از پیش در admin4b.css و admin4b.js گنجانیده شده‌اند.

CSS

تگ <link> قالب را در <head> سایت خود و پس از قالب فونت‌ها فراخوانی کنید تا فایل CSS اجرا شود.

Source code 
                            
                                <!-- link to Font Awesome CSS -->
                                <!-- link to Simple Line Icons CSS -->
                                <link rel="stylesheet" href="admin4b.min.css">

جاوا اسکریپت

تگ <script> را که در ادامه آمده است در پایان برگه‌ی خود و دقیقا پیش از تگ <body> قرار دهید. پیش‌نیازهای بوتسترپ باید پیش‌تر از پلاگین‌های جاوا اسکریپت قرار گیرند.

Source code 
                            
                                <!-- script to jQuery -->
                                <!-- script to Popper -->
                                <script src="admin4b.min.js"></script>

راه‌اندازی نخستین

نمونه‌ی کد راه‌اندازی نخستین را در سورس‌کد زیر می‌توانید مشاهده کنید.

Source code 
                            
                                <!doctype html>
                                <html lang="en">
                                  <head>
                                    <meta charset="utf-8">
                                    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
                                    <!-- link to Font Awesome CSS -->
                                    <!-- link to Simple Line Icons CSS -->
                                    <link rel="stylesheet" href="admin4b.min.css">
                                    <title>Admin 4B</title>
                                  </head>
                                  <body>
                                    <!-- script to jQuery -->
                                    <!-- script to Popper -->
                                    <script src="admin4b.min.js"></script>
                                  </body>
                                </html>
راه‌اندازی جاوا اسکریپت

همه‌ی کامپوننت‌ها به صورت خودکار توسط قالب راه‌اندازی می‌شوند.
